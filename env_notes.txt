#.env is used to store environment variables (like API keys, passwords, database URLs).

| --------------- | -------------------------------------------------------------------------------------------------------------- |
| `mysql+pymysql` | Tells SQLAlchemy: Use MySQL database with the `pymysql` Python driver                                          |
| `username`      | The MySQL database username                                                                                    |
| `password`      | The password for that user                                                                                     |
| `host`          | The server address where MySQL is running (e.g., `localhost`, `127.0.0.1`, or a Docker service name like `db`) |
| `port`          | The port MySQL is listening on (default is `3306`)                                                             |
| `moviesdb`      | The **name of the database** you want to connect to                                                            |
'''
